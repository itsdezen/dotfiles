# Global Claude Code Instructions

## Commit Style

Use emojis instead of conventional commit tags:

| Emoji | Use for |
|-------|---------|
| 🚀 | new feature / addition |
| 🐞 | bug fix |
| 🔧 | config / tooling change |
| ♻️ | refactor |
| 📝 | docs update |
| 🗑️ | remove / cleanup |
| ⬆️ | dependency update |

Example: `🚀 add smear-cursor plugin` instead of `feat: add smear-cursor plugin`
